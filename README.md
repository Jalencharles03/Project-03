# Brew Haven

Brew Haven is a fictional coffee shop website created as a final project for this course. The purpose of this project was to combine foundational web development skills with project planning and documentation practices by creating a complete website package from concept to final presentation. The website was built using HTML and CSS and focuses on creating a warm, welcoming café experience through cohesive branding, responsive layouts, and clean visual design.

The project includes:
- A responsive two-page website
- A homepage featuring Brew Haven’s featured drinks
- An About page focused on the café’s mission and community-centered atmosphere
- Project management documentation, including a scope statement, project plan, and retrospective reflection
