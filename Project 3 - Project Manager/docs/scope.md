# Project Scope Statement


## Project Purpose
The purpose of this project is to design and develop a simple website for Brew Haven, a fictional local coffee shop. The website will promote the brand, provide basic information, and demonstrate foundational web development and project management skills.
## Feedback Loops
Feedback will be gathered through peer review (Assignment 04) and instructor evaluation. Revisions will be made based on this feedback to improve clarity, design, and completeness before final submission.
## Description
This project consists of creating a two-page website for Brew Haven using HTML and CSS. The homepage will introduce the coffee shop and highlight featured items, while the about page will describe the company’s story and mission. The website will be stored in a public GitHub repository along with all required documentation.
## People Resources
- Project Manager/Developer: Myself
- Stakeholders: Instructor and peers (providing feedback)
## Desired Results
- A fully functional two-page website
- A clean and consistent visual design
- A well-organized public GitHub repository with documentation
- Demonstration of basic project planning and web development skills
## Priorities
Time: 1 Quality: 2 Budget: 3
### Time
The project must be completed by the assigned deadline of May 8, 2026
### Quality 
Quality will be measured by the following criteria:
- Both pages (index.html and about.html) load correctly without errors
- All navigation links work between pages
- The CSS file is properly linked and styles both pages consistently
- Content is clearly organized and readable
- The layout is visually clean and not broken on a standard desktop browser
- All required files are included in the GitHub repository
- The site meets all assignment requirements outlined in the project brief
### Budget
No financial budget is required; all tools and resources used will be free of cost.
## Exclusions
- No JavaScript or advanced interactivity
## Acceptance Criteria
The project will be considered complete and signed off by my course instructor, Cheri Whalen, upon final submission.
